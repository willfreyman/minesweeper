# Minesweeper (Tkinter)

Simple minesweeper game built in Python using Tkinter.

## Run

python minesweeper.py

Or run the .exe if you built one.

## Controls

- Left click: reveal a tile  
- Right click: place/remove a flag  

## Settings

Top bar lets you change:
- Rows
- Columns
- Frequency (mine density, higher number = lower difficulty)

Press "set" to apply.

## Notes

- Mine chance is based on the frequency value  
- Larger boards may slow down a bit  
- Winning just changes the reset button (no popup yet)
